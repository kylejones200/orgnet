# v0.1.0 - Initial Release

## Features

### Core Functionality
- Multi-source data ingestion (email, Slack, calendar, documents, code, HRIS)
- Graph construction with sophisticated edge weighting
- Network metrics (centrality, structural analysis)
- Community detection (multiple algorithms)
- Machine learning (GNNs, embeddings, link prediction)
- NLP analysis (topic modeling, expertise inference)
- Temporal analysis (change detection, onboarding tracking)
- Interactive visualizations and dashboards
- REST API

### Documentation
- Comprehensive examples with Jupyter notebooks
- Read the Docs setup
- Complete API reference
- User guides

### Installation

```bash
pip install orgnet
```

For optional features:
```bash
pip install orgnet[ml]    # Machine learning
pip install orgnet[viz]   # Visualization
pip install orgnet[api]   # API server
pip install orgnet[dev]   # Development tools
```

## What's New

This is the initial release of orgnet, a comprehensive platform for analyzing organizational networks using machine learning and graph analytics.
