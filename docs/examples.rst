Examples
========

The `examples <https://github.com/kylejones200/orgnet/tree/main/examples>`_ directory contains Jupyter notebooks demonstrating various features of orgnet.

Available Notebooks
-------------------

01_basic_analysis.ipynb
~~~~~~~~~~~~~~~~~~~~~~~~

Getting started with orgnet - learn the basics of loading data, building graphs, and running analysis.

02_network_metrics.ipynb
~~~~~~~~~~~~~~~~~~~~~~~~~

Deep dive into network metrics including centrality measures and structural analysis.

03_community_detection.ipynb
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Discover communities and teams using different community detection algorithms.

04_visualization.ipynb
~~~~~~~~~~~~~~~~~~~~~~

Create interactive network visualizations.

Running Examples
----------------

1. Install dependencies:

   .. code-block:: bash

      pip install faker jupyter

2. Generate sample data:

   .. code-block:: bash

      cd examples
      python generate_sample_data.py

3. Start Jupyter:

   .. code-block:: bash

      jupyter notebook

4. Open and run the notebooks in order.
